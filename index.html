Aplikasi Sistem Pendukung Keputusan (SPK) berbasis web untuk menghitung dan membandingkan alternatif terbaik menggunakan tiga metode populer.

Pilih Metode
✖️
Weighted Product
WP Method
Mengalikan nilai kriteria yang sudah dipangkatkan dengan bobot ternormalisasi. Cocok untuk data yang bersifat perkalian.
➕
Simple Additive Weighting
SAW Method
Menjumlahkan hasil perkalian bobot dan nilai ternormalisasi. Metode SPK yang paling banyak digunakan.
🎯
SMART
Simple Multi Attribute Rating Technique
Menggunakan utility function untuk menilai performa setiap alternatif berdasarkan range nilai kriteria.
Weighted Product
Metode perkalian berbobot untuk pengambilan keputusan
WP
Langkah WP:
1. Normalisasi bobot: Wj = wj / Σw
2. Hitung vektor S: Si = ∏ (xij)Wj (benefit: +Wj, cost: -Wj)
3. Hitung vektor V: Vi = Si / ΣSi
4. Alternatif terbaik = nilai V terbesar

Simple Additive Weighting
Penjumlahan terbobot nilai ternormalisasi
SAW
Langkah SAW:
1. Normalisasi matriks: rij = xij/max(xij) untuk benefit | rij = min(xij)/xij untuk cost
2. Normalisasi bobot: Wj = wj / Σw
3. Hitung nilai preferensi: Vi = Σ Wj · rij
4. Alternatif terbaik = nilai V terbesar

SMART
Simple Multi Attribute Rating Technique
SMART
Langkah SMART:
1. Tentukan bobot tiap kriteria (Rank Order Centroid / langsung)
2. Normalisasi bobot: Wj = wj / Σw
3. Hitung utility: u(xij) = (xij - min) / (max - min) × 100
4. Hitung nilai akhir: U(Ai) = Σ Wj · u(xij)
5. Alternatif terbaik = nilai U terbesar
